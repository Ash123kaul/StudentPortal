⚠️ Database Permission Required

Your Firebase database is locked. Please follow these steps to fix it:

  1. Go to Firebase Console
  2. Select your project "student-portal-cf3fa"
  3. Click on "Realtime Database" in the left sidebar
  4. Go to "Rules" tab
  5. Replace the existing rules with:
{
  "rules": {
    ".read": "auth != null",
    ".write": "auth != null"
  }
}
  • Click "Publish"
  • Refresh this page after updating the rules
